# ECサイトにおける流入経路別・売上改善分析

## 1. 分析の目的
Google Merchandise Storeの1ヶ月分のログデータを分析し、集客ルートごとの収益性を特定。広告費・集客施策の最適化に向けた意思決定を支援する。

## 2. ビジネス課題と仮説
* **課題**: 全体の訪問者数に対して購入率（CVR）が低い可能性がある。
* **仮説**: 流入数が多いチャネルにおいて、ユーザーのニーズとサイトの提供価値にミスマッチが生じているのではないか。

## 3. 分析プロセスと結果

### ステップ1：チャネル別売上集計
全流入経路を比較した結果、以下の傾向が判明。
* **YouTube**: 訪問者数は180名と多いが、売上は0ドル。
* **メール(Gmail)**: 訪問者は2名だが、1件の成約があり、最も成約率が高い。

### ステップ2：YouTubeユーザーの行動深掘り
YouTube経由ユーザーの質を評価するため、回遊性を分析。
* **平均ページビュー**: 1.66
* **平均滞在時間**: 190.9秒（約3分）

#### 【洞察】
ユーザーは1ページあたりの滞在時間は長い（約3分）が、2ページ目へ遷移せずに離脱している。これは「動画内容には興味を持ってサイトに来たが、次に何をすべきか（購入導線）が不明確である」ことを示唆している。

## 4. 具体的提案（Actionable Insights）
1. **YouTube専用LPの導入**: 動画で紹介した商品をトップに配置し、購入ボタンを強調した専用ランディングページを用意する。
2. **メールマーケティングの強化**: 高い成約率を誇るメール経由の流入を増やすため、既存顧客リストへのアプローチ優先度を上げる。

## 5. 使用したSQLクエリ（一部抜粋）

### ユーザー行動分析クエリ
```sql
SELECT 
    trafficSource.source AS source, 
    AVG(totals.pageviews) AS avg_pageviews, 
    AVG(totals.timeOnSite) AS avg_time_on_site
FROM 
    `bigquery-public-data.google_analytics_sample.ga_sessions_*`
WHERE 
    _TABLE_SUFFIX BETWEEN '20170801' AND '20170831' 
    AND trafficSource.source = 'youtube.com'
GROUP BY 
    source;
